# Man Buys Jewellery Website

Landing page and marketing site for Man Buys Jewellery.

## Pages

- **Landing** — Hero, CTA to app
- **Features** — Feature highlights  
- **Testimonials** — User testimonials
- **Navigation** — Links to app, features, testimonials

## Branding

- **Colors**: Navy (#1F3557), Gold (#D4A574), Pink accent (#F5D7E3)
- **Typography**: Inter (modern, readable)
- **App Link**: https://man-buys-jewellery-eight.vercel.app

## Development

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deployment

Deployed to Vercel. Push to main branch for automatic deployment.

```bash
npm run build
npm start
```
